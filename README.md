## フレームワーク

## DB

* PostgreSQL

## 開発環境

## アプリ仕様

* 顧客から受け付けた問い合わせの一覧表示を行う
* 案件名,顧客名（入力式）での検索が可能
* ステータス,種別,担当者（選択式）での絞り込みが可能

* 以下は追加予定の機能
　　 * ページネーション
　　 * お気に入り登録
